# Maiteth Jones Portfolio

Welcome to my portfolio website! This project showcases my skills and projects, providing a glimpse into my journey as a Front End Developer.

## Pages

### Home

The Home page features an interactive canvas with particles that move around, creating a visually engaging experience. The canvas is implemented using JavaScript with various canvas functions.

### About Me

The About Me page provides information about who I am, my skills, and background. It's a text-based page designed to introduce visitors to the person behind the projects.

### Projects

The Projects page is currently under construction but includes a "Coming Soon" message with stylish CSS effects. Stay tuned for updates as it will soon showcase a collection of my projects and achievements.

### Contact

The Contact page includes a form with a unique design implemented using CSS. Visitors can use this form to send messages directly to my Gmail account.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository.
2. Open the `index.html` file in your browser to explore the portfolio.
